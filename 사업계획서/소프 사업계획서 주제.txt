무작위 진법 암호화 프로그램

암호화할 데이터 입력(영어, 숫자) : asdvasdsdf -> 띄어쓰기 가능하도록 해서 토큰 뽑아서 리스트에 넣자.
변환할 진법 무작위로 선정 import random

각 문자를 유니코드 값으로 변환하고 선택된 진법으로 변환한다.
변환된 문자열은 띄어쓰기 없도록.

즉 띄어쓰기를 구별할 방법도 만들어야 함. 각 단어 길이가 가장 유력.
띄어쓰기도 코드가 있으니까 한꺼번에 변환할까?

ex) suction 2735
-> 73 75 63 74 69 6f 6e 20 32 37 33 35 (16진수)

암호화 과정
평문을 유니코드로 변환
각 코드를 n진수로 암호화 (직접 프로그램으로 구현)
암호화된 코드를 유니코드 문자로 출력 (암호문)

key도 랜덤으로 생성. 이걸 전달해야 함.

복호화 과정
암호화된 문자열을 다시 유니코드 로 변환
코드를 n진수로 복호화 ( int(str, n) ) 
복호화된 코드를 유니코드 문자로 다시 출력 (평문)
