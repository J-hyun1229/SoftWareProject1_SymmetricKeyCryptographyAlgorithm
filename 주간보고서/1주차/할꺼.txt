keyString = "" -> 암/복호화용 키, 문자열
conversionNum = 0 -> 변환할 진법. int형.

주간보고서 구성 짜기

형식 구성
-> ppt로 적어서 pdf로 내거나
-> 한글파일 샘플 양식대로 적어서 내거나

내용 구성
1. 이번주차 계획

2. 진행 상황(주요 내용)

3. 다음주차 예정 계획

4. 첨부자료 (선택사항)

5주차에 걸쳐 프로젝트 진행
주별 진행계획

1주차 - 사용할 라이브러리, 변수 타입, 프로그램 구성 등 뼈대작업 (list, tuple, dictionary까지 배움)
2주차 - 암호화 프로그램 초안 작성 (유니코드로 변환, 진법연산, 다시 문자열로 변환) (String 배움)
3주차 - 복호화 프로그램 초안 작성 (함수만들기 배움)
4주차 - 셸에서 작동하지 않고 파일 입출력으로 발전시키기 (파일 입출력 배움)
5주차 - 프로그램에 객체지향 개념 추가해서 코드 개선하기 (객체지향 개념 배움)
