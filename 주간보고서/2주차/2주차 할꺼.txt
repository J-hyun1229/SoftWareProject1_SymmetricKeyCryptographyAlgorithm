2주차 암호화 프로그램 작성하기

암호화 프로그램 작동과정

1. 원본 데이터 입력
-> input()

2. 원본 데이터를 유니코드로 변환
-> 각 문자마다 변환. 배열로 처리. ord() 이용.

3. 유니코드로 변환한 데이터를 랜덤 진법으로 변환.
-> 유니코드 리스트의 각 요소마다 진법연산.

4. 키를 생성.
-> 전환된 리스트와 같은 길이의 리스트로 랜덤 생성.
-> random 라이브러리 함수 이용.

5. 진법변환된 코드와 키를 XOR연산.
-> ^ 이용.

6. 진법변환된 코드를 다시 문자열로 변환.
-> chr() 이용.
